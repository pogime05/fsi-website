# FSI Freight Solutions Website

A polished static starter website for FSI Freight Solutions, built with HTML, CSS, JavaScript, the company logo, and a project-local hero image.

## Open locally

Open `index.html` in a browser.

## Files

- `index.html` - page structure and content
- `styles.css` - responsive styling
- `script.js` - header scroll state and demo form behavior
- `assets/fsi-logo.png` - cropped company logo for website use
- `assets/fsi-logo-original.jpg` - original logo file copied into the project
- `assets/hero-freight.png` - generated hero image used on the homepage
